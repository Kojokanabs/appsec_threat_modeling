# PayNow – Threat Modeling Assignment

Threat modeling workshop for a payment application focusing on payment processing and card storage.

## Scope
- Web/Mobile Client
- API Gateway
- Payment Service
- Database
- Third-party Payment Processor

## Methodology
- STRIDE for threat identification
- Risk assessment
- Security mitigations & controls

## Objective
Identify realistic threats early and propose practical security controls.

For educational and secure design practice.
